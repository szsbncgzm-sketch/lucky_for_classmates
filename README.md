# 运行并部署你的 AI Studio 应用

这里包含你在本地运行应用所需的一切。

在 AI Studio 中查看你的应用：https://ai.studio/apps/ccfb539a-3e36-4746-9649-1fb23d11c656

## 本地运行

前置条件：Node.js

1. 安装依赖：
   `npm install`
2. 将 [.env.local](.env.local) 文件中的 `GEMINI_API_KEY` 设置为你的 Gemini API 密钥
3. 运行应用：
   `npm run dev`
